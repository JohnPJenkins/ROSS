12 /**
13  * \brief A Friis propagation loss model
14  *
15  * The Friis propagation loss model was first described in
16  * "A Note on a Simple Transmission Formula", by
17  * "Harald T. Friis".
18  *
19  * The original equation was described as:
20  * \f$ \frac{P_r}{P_t} = \frac{A_r A_t}{d^2\lambda^2} \f$
21  * with the following equation for the case of an
22  * isotropic antenna with no heat loss:
23  * \f$ A_{isotr.} = \frac{\lambda^2}{4\pi} \f$
24  *
25  * The final equation becomes:
26  * \f$ \frac{P_r}{P_t} = \frac{\lambda^2}{(4 \pi d)^2} \f$
27  *
28  * Modern extensions to this original equation are:
29  * \f$ P_r = \frac{P_t G_t G_r \lambda^2}{(4 \pi d)^2 L}\f$
30  *
31  * Here, we ignore tx and rx gain and the input and output values are in dB or dBm:
32  * \f$ RX_{dB} = TX_{dB} + 10\mathrm{log_{10}} \frac{\lambda^2}{(4 \pi d)^2 L}\f$
33  *
34  * With:
35  * - \f$ P_r \f$ : reception power (W)
36  * - \f$ P_t \f$ : transmission power (W)
37  * - \f$ G_t \f$ : transmission gain (unit-less)
38  * - \f$ G_r \f$ : reception gain (unit-less)
39  * - \f$ \lambda \f$ : wavelength (m)
40  * - \f$ d \f$ : distance (m)
41  * - \f$ L \f$ : system loss (unit-less)
42  *
43  *
44  * This model is invalid for small distance values.
45  * The current implementation returns the txpower as the rxpower
46  * for any distance smaller than MinDistance.
47  */
