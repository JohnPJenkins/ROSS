Detailed Description

SRW module. Contains functions relevant / necessary for the functional requirements pertaining to the SRW waveform. NOTE: None of the netdmf_* event handlers below generate artificial data – we assume that the NetDMF file was read in and that no other data should be generated that was not in that file.
