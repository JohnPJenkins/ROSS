219 /**
220  * OSPF model LP init function.
221  *
222  * This is the OSPF model LP initialization function. This function is
223  * responsible for initializing each OSPF LP. The main steps are:
224  * <ol>
225  * <li>Create and init the neighbor array
226  * <li>Create and init the LP LSA database
227  * <li>Create and init the LP forwarding table
228  * <li>Start the aging timer for the oldest LSA
229  * </ol>
230  *
231  * The order of steps 2 and 3 are dependant upon if the model is started in
232  * converged state or not. The determination is based on the fact that
233  * sometimes we build the forwarding table from the LSA database, and sometimes
234  * we read it in from file.
