24 /**
25  * \brief A Friis propagation loss model
26  *
27  * The Friis propagation loss model was first described in
28  * "A Note on a Simple Transmission Formula", by
29  * "Harald T. Friis".
30  *
31  * The original equation was described as:
32  * \f$ \frac{P_r}{P_t} = \frac{A_r A_t}{d^2\lambda^2} \f$
33  * with the following equation for the case of an
34  * isotropic antenna with no heat loss:
35  * \f$ A_{isotr.} = \frac{\lambda^2}{4\pi} \f$
36  *
37  * The final equation becomes:
38  * \f$ \frac{P_r}{P_t} = \frac{\lambda^2}{(4 \pi d)^2} \f$
39  *
40  * Modern extensions to this original equation are:
41  * \f$ P_r = \frac{P_t G_t G_r \lambda^2}{(4 \pi d)^2 L}\f$
42  *
43  * Here, we ignore tx and rx gain and the input and output values are in dB or dBm:
44  * \f$ RX_{dB} = TX_{dB} + 10\mathrm{log_{10}} \frac{\lambda^2}{(4 \pi d)^2 L}\f$
45  *
46  * With:
47  * - \f$ P_r \f$ : reception power (W)
48  * - \f$ P_t \f$ : transmission power (W)
49  * - \f$ G_t \f$ : transmission gain (unit-less)
50  * - \f$ G_r \f$ : reception gain (unit-less)
51  * - \f$ \lambda \f$ : wavelength (m)
52  * - \f$ d \f$ : distance (m)
53  * - \f$ L \f$ : system loss (unit-less)
54  *
55  *
56  * This model is invalid for small distance values.
57  * The current implementation returns the txpower as the rxpower
58  * for any distance smaller than MinDistance.
59  *
60  * MinDistance is defined as follows: (Paul, Whites, Nasir, Intro. to Electromagnetic Fields)
61  * \f$ d = \frac{kD^2}{8\lambda} \f$
62  * Picking a typical value of 16 for k, and assuming a 1/4 wavelength antenna MinDistance becomes:
63  * \f$ d = \frac{1}{8}\lambda \f$
64  *
65  *
66  */
